# CodeAlpha Basic Network Sniffer

## Description

This project is a basic network packet sniffer developed using Python and Scapy.

## Features

* Captures network packets
* Displays packet summaries
* Shows source and destination information
* Demonstrates network traffic monitoring

## Technologies Used

* Python 3.14
* Scapy
* Npcap

## How to Run

1. Install Python
2. Install Scapy:
   py -m pip install scapy
3. Install Npcap
4. Run:
   py sniffer.py
